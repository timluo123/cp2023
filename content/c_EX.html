int 測試（int x，int y）
{
    整數 n = 100；// 定義變數 'n' 並為其賦值 100
    int val = abs(x - n); // 計算 'x' 和 'n' 之間的絕對差並將其儲存在 'val' 中
    int val2 = abs(y - n); // 計算 'y' 和 'n' 之間的絕對差並將其儲存在 'val2' 中

    // 檢查「val」是否等於「val2」。如果為 true，則傳回 0。否則，如果 'val' 小於 'val2'，則傳回 'x'，否則傳回 'y'。

}
